Analyze the Apache-style access log located at `/app/access.log` and summarize the request statistics by writing a JSON report to `/app/report.json`.

The JSON report must contain the following keys:
1. "total_requests": The total number of requests in the log file.
2. "unique_ips": The number of unique client IP addresses.
3. "top_path": The most frequently requested HTTP path.

You have 120 seconds to complete this task. Do not cheat by using online solutions or hints specific to this task.
